3. The changes we have made

We have not changed much from the original design we detailed in P2 because we planned our design and the sites functionallity well.

Changes to all websites: Istead of having a roll down menu, we added all the links as divs in a row, but added a rolldown menu when the veiwport width is under 1000px so that the site is more reactive. We also changed the design on the images and rounded them off instead of having them display as squares which was the original idea

Changes to hybler.html: Decided to not make individual websites for the houses that are displayed in hybler.html and instead made a slideshow of the images in hybler.html

Changes to tilbud.html: We decided to make the onhover elements on the page display divs using Javascript in stead of CSS which was our original plan. We also decided to only make one option for tilbud.html. The original idea was to include more information about other supplies that SiT offered, but we were short on time and prioritised making the website more reactive and user-friendly over informatio on the website.

Changes to hybler.html: Since we added another block of Javascript to tilbud.html we decided to ditch making a favourite selector in hybler.html because we were running out of time and prioritized polishing the website

General changes from the Generic Features mockup Text-color is black instead of dark gray (#737373). Body width is not set at 960px, its 100% and the content width varies from page to page. We did not include the changing title tab for each page, which could be seen on the mockups.

Kontakt oss: In the mockups we provided an open space for a picture of the developers of this webpage. In the end we decided to not include the picture, and instead clean and simple page with easily accessable contact information about SiT.
